CSVDecoder

public final class CSVDecoder

This csv decoder is narrowly designed to support decoding comma-delimited files where the format of the string values are wrapped with “” marks and the file includes a single line table header of key names.

Decoding Values
  • decodeArray(_:from:)

    Decodes a top-level value of the given type from the given CSV representation.

    Note

    It is assumed that the csv is formatted with a single header row with the keys for each column.

    Throws

    DecodingError.dataCorrupted if values requested from the payload are corrupted, or if the given data is not valid CSV.

    Throws

    An error if any value throws an error during decoding.
